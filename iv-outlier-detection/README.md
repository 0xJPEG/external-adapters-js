# Chainlink External Adapter for iv-outlier-detection

## Input Params

- `base`, `from`, `coin` or `symbol`: The symbol of the currency to query
- `days` or `period`: The number of days to get the IV result from

## Output

```json
{
   "jobRunID":"1",
   "data":{
      "result":60.1
   },
   "result":60.1,
   "statusCode":200
}
```
