# Chainlink External Adapter for EthGasStation

## Input Params

- `speed`: The speed for gas price to get (required). Available choices:
    - `safeLow`
    - `average`
    - `fast`
    - `fastest`
- `endpoint`: The endpoint to use (optional, default: ethgasAPI)
